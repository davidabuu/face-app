Welcome To Afe, Babalola University Dress Code Detection System


Enforcing this policy is a serious challenge in academic institutions which creates the need for system to do so using AI techniques .

What We Offer

It is a system that checks if students are properly dressed in accordance to the university's dress code guidelines or policies. It checks for specific features or objects on students when they are in front of the system and the detect button is clicked by the student. It displays whether the student is properly dressed or not and displays what the student did wrong if not properly dressed.
